## MDN 切换中文
### 方法一：修改 URL
将 URL 中的 en-US 换成 zh-CN。
```html
https://developer.mozilla.org/en-US/docs/Learn/CSS/Styling_boxes/Backgrounds
```

### 方法二：语言菜单
通过页面右上方 `LANGUAGES` 菜单进行切换
