# 数组 Array

```javascript
var skills = ['卖萌', '碎觉', '瞪眼', '三分球', '各种语言的 Hello World'];
```

## 常用 API
`Array.prototype.push()`
```javascript
> var skills = ['卖萌', '碎觉', '瞪眼', '三分球', '各种语言的 Hello World'];
→ undefined

> skills.push('自由式蛙泳')
→ 6

> skills
→ ["卖萌", "碎觉", "瞪眼", "三分球", "各种语言的 Hello World", "自由式蛙泳"]
```
