# 循环

## while
`while` 在条件表达式为**真**的情况下循环执行一段代码，条件表达式会在代码块被执行前求值。
```javascript
//
var n = 1;
while (n <= 10) {
  console.log(n);
  n += 1;
}
```

```javascript
while (true) {
  // 一直不断被执行的代码
}
```

```javascript
while (false) {
  // 永远不会被执行的代码
}
```
