# 字符串 String

## 声明字符串
字符串可以用一对双引号 `"` 或者单引号 `'` 括起来。
```javascript
"但愿人长久，千里共婵娟。"

'其实我是一个演员。'
```

## 拼接字符串
可以通过 `+` 拼接字符串，如
```javascript
> "唧唧复唧唧，" + "木兰当户织。"
→ "唧唧复唧唧，木兰当户织。"
```


## 字符串长度 length
```javascript
> "Are you OK?".length
→ 11
> "举头望明月，低头思故乡。".length
→ 12
> "好好Study，天天Happy。".length
→ 16
```
