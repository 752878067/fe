## 拼接字符串
可以通过 `+` 拼接字符串，如
```javascript
> "唧唧复唧唧，" + "木兰当户织。"
→ "唧唧复唧唧，木兰当户织。"
```


## 字符串长度 length
```javascript
> "Are you OK?".length
→ 11
> "举头望明月，低头思故乡。".length
→ 12
> "好好Study，天天Happy。".length
→ 16
```
