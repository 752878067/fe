# AJAX

AJAX 全称 Asynchronous JavaScript and XML，是一种通过**异步**请求，**无需刷新**实现页面局部更新的技术。

## XMLHttpRequest
`XMLHttpRequest` 简称 `XHR`，通常用于发送 HTTP 请求。
