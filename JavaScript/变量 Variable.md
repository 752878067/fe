# 变量 Variable

变量是一块用于存储值的内存空间，变量的特别之处在于它的值可以被改变。

## 声明变量
声明变量同时赋值
```javascript
var name = '樱木花道';
var age = 16;
```

先声明变量再赋值
```javascript
var sc;  // → undefined
sc = 'Stephen Curry';
```

更改变量的值
```javascript
var price = 1000;
price = 1500;
```



## 变量名
```javascript
var hero name       // ❌ 变量名不能包含空格
var 2b              // ❌ 变量名不能以数字开头
var product_price   // 🆗
var $container      // ✅ 
var _counter        // ✅
var observable$     // ✅ 
var firstName       // ✅ 💯
var isMobileSafari  // ✅ 💯
```
