# 运算符 Operator

## 算术运算符

| 运算符 | 操作 | 🌰        |
|:-----:|-----|-----------| 
|  `+`  | 加  | `3 + 5`   |
|  `-`  | 减  | `20 - 7`  |
|  `*`  | 乘  | `4 * 8`   |
|  `/`  | 除  | `100 / 8` |

```javascript
3 + 5 * 6

(2 - 8) * 5

(100 + 53) / 7
```
