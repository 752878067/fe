# 运算符 Operator

## 算术运算符

| 运算符 | 操作 | 🌰            |
|:-----:|:---:|---------------| 
|  `+`  | 加  | `3 + 5`       |
|  `-`  | 减  | `20 - 7`      |
|  `*`  | 乘  | `4 * 8`       |
|  `/`  | 除  | `100 / 8`     |
|  `%`  | 模  | `50 % 7` 得 1 |

```javascript
> 2 + 8
→ 10

> 9 - 3
→ 6

> 3 * 7
→ 21

> 10 / 8
→ 1.25

> 67 % 10
→ 7

> (3 - 5) * 8
→ -16

> 5 * 8 % (9 - 2)
→ 5

> -3 * 7 - -5 * 8
→ 19
```

## 比较运算符
```javascript
> 8 > 6
→ true

> 5 < -2
→ false

> 6 == 7
→ false

> 99 != 100
→ true

> 99.99 >= 99.99
→ true

> 2.3 <= -3
→ false
```
