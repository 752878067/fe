# 事件细节 Event Details

## 鼠标事件
* `click`
* `dbclick`
* `contextmenu`
* `mousedown`
* `mouseup`
* `mousemove`
* `mouseover`
* `mouseout`

事件顺序 `mousedown` → `mouseup` → `click`

窗口坐标 `event.clientX` `event.clientY`

文档坐标 `event.pageX` `event.pageY`

键盘修饰键 `event.altKey` `event.ctrlKey` `event.shiftKey` `event.metaKey`(Mac Command 键)

无选中效果 `user-select: none`
