# Sass 简介

Sass(Syntactically Awesome Style Sheets) 是一种 CSS 预编译器，它通过提供一些 CSS 不具备的特性，如嵌套，循环，函数等，使样式文件更加模块化和易于维护，并最终编译成传统的 CSS 文件用于部署。

## 安装

## 嵌套
```sass
.scene {
  perspective: 1000px;
  .cube {
    width: 200px;
    height: 200px;
    position: relative;
    transform-style: preserve-3d;
    .face {
      top: 0;
      left: 0;
      width: inherit;
      height: inherit;
      position: absolute;
    }
  }
}
```
⚠️ 尽量控制嵌套数在 4 层以内。

## & 引用
```scss
a {
  color: #555;
  &:hover {
    color: #333;
  }
  &:active {
    color: #444;
  }
}

.flex-item {
  .no-flexbox & {
    float: left;
  }
}

.container {
  > .navbar {
    .header > & {
      background: deepskyblue;
    }
  }
}
```

## 注释
单行注释 `//` 编译后不会出现在 CSS 里。
```scss
// Header
.header {
  height: 60px;
}
```
