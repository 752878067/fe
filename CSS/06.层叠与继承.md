# 层叠与继承

## 层叠
### !important
```html
<div class="success no-background-color" id="faces">
  😀😁🤣😂😄😅😆😇😍😌😋☺️🙃🙂😊😁😀😉🤡🤠😏😶😑😒🙄😎🤗🤔
</div>
```
```css
#faces {
  background-color: aqua;
}

.success {
  background-color: green;
}

.no-background-color {
  background-color: transparent !important; /* ✌️ */
}
```

### Specificity 特异性
|分数  | 条件 |
|----:|------|
| 1000| + 1 / 在 `style` 属性内或在 `<style>`标签内 |
| 100 | + 1 / 每个出现的 ID 选择器 |
| 10  | + 1 / 每个出现的 类，伪类或 属性选择器 |
|  1  | + 1 / 每个元素选择器或伪元素 |
