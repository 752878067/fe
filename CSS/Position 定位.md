# Position 定位

## 正常文档流 Normal Flow
Normal Flow 即浏览器默认的文档布局方式。

定位就是通过设置 `position` 属性的值来覆盖默认的布局方式。

## 静态定位
`position: static;`

## 相对定位
`position: relative;`

## 绝对定位
`position: absolute;`

## 固定定位
`position: fixed;`
