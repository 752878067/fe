## 外边距合并
在正常流情况下，外边距会进行合并。
```html
<h1>好好学习</h1>
<h2>天天向上</h2>
```
```css
h1 { margin: 20px 0; }
h2 { margin: 30px 0; }
```

## 外边距为负
```html
<div class="t"></div>
```
```css
.t {
  width: 30px;
  height: 200px;
  margin: 50px auto;
  background: orange;
}

.t::after {
  content: '';
  height: 30px;
  margin: 0 -80px;
  display: block;
  background: limegreen;
}
```
**运行结果** https://codepen.io/twhy/pen/zzrQMR
