# CSS 应用

## 简单示例
```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>CSS</title>
    <link rel="stylesheet" href="style.css">
  </head>
  <body>
    <h1>Hello CSS!</h1>
    <p>CSS 很 666666……</p>
  </body>
</html>
```
```css
h1 {
  padding: 10px;
  background-color: orange;
  border: 1px solid black;
}

p {
  color: purple;
}
```
