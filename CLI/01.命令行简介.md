# 命令行简介.md

命令行是一种与图形界面相对的，通过键入命令对计算机执行操作的一种界面。

**注意** 目录就是平常说的文件夹

## 基本命令
### pwd
显示当前所在目录
```bash
$ pwd
/Users/twhy
```

### ls
列出当前目录内容
```bash
$ ls
Applications  Documents	 Music  Code
```
### cd
返回当前用户目录
```bash
$ cd
$ pwd
/Users/twhy
```
返回上级目录
```bash
$ cd ..
```
进入特定目录，如 `Code/Github`
```bash
$ cd Code/Github
```
